# SnapTools — Free Online Tools

A collection of 20 free online tools built with React, Vite, and Tailwind CSS.

## Tools

### Developer Tools
- JSON Formatter & Validator
- Base64 Encoder/Decoder
- URL Encoder/Decoder
- CSS Minifier
- HTML Entity Encoder/Decoder
- Markdown to HTML Converter
- Hash Generator (MD5, SHA-1, SHA-256)
- QR Code Generator
- UUID Generator
- Timestamp Converter

### Text Tools
- Word & Character Counter
- Lorem Ipsum Generator
- Case Converter
- Password Generator

### Design Tools
- Colour Converter (HEX/RGB/HSL)
- Colour Palette Generator
- Image Resizer

### Social Media
- Instagram Font Generator

### Finance
- UK Mortgage & Stamp Duty Calculator
- Invoice Generator

## Getting Started

```bash
# Install dependencies
bun install

# Development
bun run dev

# Build
bun run build

# Preview build
bun run preview
```

## Firebase Deployment

```bash
# Install Firebase CLI
npm install -g firebase-tools

# Login
firebase login

# Init (select Hosting, use existing project)
firebase init hosting

# Deploy
firebase deploy
```

## Tech Stack

- **React 19** + **TypeScript**
- **Vite 8** (build tool)
- **Tailwind CSS v4** + **shadcn/ui**
- **Convex** (backend)
